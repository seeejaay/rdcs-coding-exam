
# Coding Exam

Create a Laravel backend:
- CRUD API Endpoint for **USERS**
- CRUD API Endpoint for **ROLES**
- API Endpoint for **LOGIN**

Create a ReactJS frontend:
- Simple Login Form
- HomePage should display:
    - Full Name of logged-in user
    - Logout Button
- CRUD Operations for **ROLE**
- CRUD Operations for **USERS**

To Migrate Database:
```php 
php artisan migrate
```
Install Laravel Dependencies:
```php
composer Install
```

Run Laravel:
```php
php artisan serve
```
Install ReactJS Dependencies:
```js
npm install 
```

Run ReactJS:
```js
npm run dev or npm run build
```
